## ğŸš€ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A local web server (for running HTML applications)

### Running the Applications

For the interactive applications (hangman, notes-app, todo-app), you'll need to serve the files through a local web server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser and navigate to the desired application.

### Running Individual Scripts

Most JavaScript files can be run directly with Node.js:

```bash
node basics/hello-world.js
node functions/closures.js
```
